Данный чат-бот был создан для изучения английского языка. Данный чат-бот каждый день рассылает пользователям слова.
Пользователи учат слова и затем сдают их боту.
В боте реализована система интервального повторения и он напоминает пользователям слова.

В боте используется работа с СУБД postgresql, с библиотеками random, vk-api, json, time, multiprocessing, psycopg2.
